
# Modern Quiz Platform

This project is a modern, aesthetic, and innovative quiz platform built with Next.js, TailwindCSS, Zustand, and Framer Motion. It features randomized questions, animated sidebar navigation, instant feedback, responsive design, dark/light mode, confetti effects, and more.

## Features
- Randomized questions on each visit
- Animated sidebar navigation
- Answer selection with instant feedback
- Optimized rendering for large quizzes
- Responsive, modern UI (glassmorphism/neumorphism)
- Smooth animations and transitions
- Progress bar
- Dark/light mode toggle
- Save progress in localStorage
- Confetti on correct answer
- Timer per question
- End screen with score breakdown and retry

## Tech Stack
- Next.js
- TailwindCSS
- Zustand
- Framer Motion

## Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Accessibility
- Keyboard navigation
- ARIA roles
- Color contrast

## Folder Structure
- `src/components/` - UI components
- `src/store/` - Zustand state management
- `src/utils/` - Utility functions
